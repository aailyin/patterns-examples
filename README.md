# Patterns on JavaScript

Just a repo with several examples of popular patterns written to practise myself.
Based on examples from course https://github.com/YauhenKavalchuk/design-patterns

Patterns: 
* Behaviour:
  * Mediator
* Creational:
  * Abstract factory
  * Bridge
  * Builder
  * Factory method
  * Prototype
  * Singleton
* Structural: 
  * Adapter
  * Composite
  * Decorator
  * Facade
  * Flyweight
  * Proxy
